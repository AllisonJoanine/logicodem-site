# LogiCodem Solutions - Website

Site profissional e responsivo da LogiCodem Solutions, empresa especializada em desenvolvimento de software, sistemas web, automações e integrações de APIs.

## Características

- **Design Moderno e Minimalista**: Interface limpa com paleta de cores profissional (azul escuro, branco e dourado)
- **Totalmente Responsivo**: Funciona perfeitamente em desktop, tablet e dispositivos móveis
- **Performance Otimizada**: Carregamento rápido e otimizado para SEO
- **Animações Suaves**: Efeitos visuais elegantes e transições fluidas
- **Menu Mobile**: Navegação intuitiva para dispositivos menores
- **Formulário de Contato**: Integração com WhatsApp para facilitar comunicação

## Estrutura do Projeto

```
logicodem-site/
├── index.html          # Arquivo HTML principal
├── css/
│   └── styles.css      # Estilos CSS completos
├── js/
│   └── script.js       # Funcionalidades JavaScript
├── images/             # Imagens e assets
├── README.md           # Este arquivo
└── .gitignore          # Arquivo de configuração Git
```

## Seções do Site

1. **Header**: Logo, menu de navegação e links rápidos
2. **Hero**: Seção principal com chamada para ação
3. **Serviços**: Apresentação dos 4 principais serviços
4. **Sobre**: Informações sobre a empresa e seus valores
5. **Contato**: Formulário e informações de contato
6. **Footer**: Informações adicionais e direitos autorais

## Tecnologias Utilizadas

- HTML5
- CSS3 (com variáveis CSS e Grid/Flexbox)
- JavaScript (Vanilla)
- Font Awesome (ícones)
- GitHub Pages (hospedagem)

## Como Usar Localmente

1. Clone o repositório
2. Abra o arquivo `index.html` em seu navegador
3. Ou use um servidor local:
   ```bash
   python -m http.server 8000
   ```
